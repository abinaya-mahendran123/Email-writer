📧 **Mail Maestro**

Mail Maestro is an AI-powered email generation application built with Streamlit and Google's Gemini AI. The application helps users quickly draft professional emails by providing a topic or title. It automatically generates a complete email including the subject, body, signature, and appropriate tone.

🚀** Features**
Generate emails instantly from a topic
AI-powered email drafting using Gemini 2.5 Flash

**Structured email format:**
Subject
Email Body
Signature
Tone-adjusted content generation
Simple and user-friendly interface

🛠️ **Technologies Used**
Python
Streamlit
Google Generative AI (Gemini API)
